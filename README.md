# technologytriviaquiz
The ultimate best UI trivia quiz for technology enthusiasts!

**How to play the game:**

1. Download all files from the repository (or from the "Released binaries" list)
2. Extract the .tar file
3. Head to the folder containing the extracted files.
4. Run the main.py file and begin the game (make sure to have Python installed in your OS or IDE):

*Note: for those who want to run their files inside their OS, you may need to follow the guidelines (requirement: Python must be INSTALLED!):*

*For Windows:*
```
python main.py
```
*For all Linux distributions:*
```
python3 main.py
```
